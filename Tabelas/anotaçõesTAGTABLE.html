TAG TABLE A tag table sozinha não faz nada, ela apenas separa um espaço no seu HTML para inserirmos dados tabelares. tag tr: Representa a linha da tabela, vai dentro da tag table. tag td: representa a coluna da tabela, vai dentro da tag tr. Podemos colocar os atributos na tag table para modificarmos nossa tabela. Atributo border: Cria uma borda na tabela. Atributo width: Aumenta a largura da tabela. Atributo Align: Podemos alinhar a tabela a direita, esquerda ou no centro por exemplo. Atributo cellspacing: Aumenta ou diminui o espaçamento entre as celulas. Atributo cellpadding: Aumenta ou diminui o espaçamento do padding da celula. Atributo summary: Atributo usado para o leitor de tela poder ler a informação da tabela que colocarmos.
