TAG INPUT E SEUS TIPOS tag input Utilizado para criação de campos onde iremos inserir dados. types do imput text: cria um campo texto comum, para inserirmos nomes, números, caracteres... number: cria um campo para inserirmos apenas números, como idade por exemplo. aqui podemos determinar um número mínimo e um máximo, utilizando os atributos "min" e "max". Utilizando o atributo "step", podemos dar a opção de o usuário ir pulando os números. Ex: step="5" a cada clique na seta, irá aumentar ou diminuir em 5 o campo número. button: Cria um botão onde possui algumas funções. Obs: Semanticamente, é melhor criar a tag button e não a input criando um type="button. o que colocarmos no valor do atributo "Value" é o que vai aparecer escrito dentro do botão. range: Com ele criaremos um campo scroll, onde podemos definir valores. Ex: min="0" max="100" E ainda podemos colocar ele pra já começar com um valor, colocando o atributo "value="Valor desejado"". Color: Cria um botão com uma caixa de diálogo de cor, onde podemos escolher cores. email: Cria um campo para inserirmos um e-mail. Url: Campo criado para inserimos uma url. Date: Campo criado para colocarmos uma data. Week: Cria um calendário para selecionarmos uma semana do ano. Obs: Só funciona no Chrome e no Edge. Month: Cria um calendário para selecionarmos o mês e o ano. Obs: Só funciona no Chrome e no Edge. Checkbox: Cria-se uma caixinha de seleção. Usa-se muito também com valores booleanos... True or False, 0 ou 1... Aqui pode-se selecionar quantas caixinhas quiser, quantas alternativas quiser. Radio: Cria-se uma caixa de seleção, que ao determinar o atributo "name" com os mesmos valores, só poderá ser selecionado uma caixa. Hidde: É um campo que fica invisível para o usuário. File: Campo criado para o usuário enviar um arquivo. Colocando o atributo multiple também, o usuário consegue selecionar vários arquivos para enviar de uma vez. Search: Campo de texto feito para busca, possui um botão x onde pode limpar o que foi escrito. Obs: Só funciona no Chrome e no Edge.
