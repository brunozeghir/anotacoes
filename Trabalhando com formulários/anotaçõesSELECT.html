SELECT É uma lista pré definida de alguns valores que você quer que o usuário escolha. A gente cria uma caixa, onde o usuário irá selecionar as opções que damos. Se colocarmos o atributo "selected" em uma opção, ele já carrega para o usuário com essa opção selecionada. O atributo "multiple" faz com que o usuário possa selecionar mais de uma opção apertando o ctrl no teclado.
