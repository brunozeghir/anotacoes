BUTTON E SEUS TIPOS No button podemos usar o javascript por exemplo para criarmos interações ao clicarmos nele. Por exemplo: onmouseover="alert('Passei aqui')", aqui ao passar o mouse em cima do botão, irá aparecer um aviso na tela com a frase "Passei aqui". type button: É o botão mesmo, tipo mais simples de botão. type reset: O botão ao ser clicado, irá limpar os campos do formulário que ele está dentro. type submit: É o tipo de botão que envia o formulário preenchido.
