AULA TAG FORM tag form Usada para criar formulários, dentro dela estará tags como input, button... entre outras. tag imput tag utilizada para criação de campos de preenchimento do usuário. Possui types (atributos), q dependem da necessidade daquele campo. Ex: type="text" abre um campo para preenchimento de texto. type="number" abre um campo para preenchimento somente de números. type="passsword" abre um campo para preenchimento de senhas, onde o que é escrito não aparece na tela. tag button tag utilizada para criação de botões. atributo name serve para nomearmos um campo, que será necessário identificarmos posteriormente. atributo action É colocado o endereço do servidor. Direciona pra onde o formulário irá. atributo method possui dois valores, o "GET" e o "POST" get = Manda as informações preenchidas no formulário pela URL do navegador. (Não seguro) POST = Enviar uma requisição para o servidor (Método seguro, usar este.) atributo target Atributo que possui os valores "_blank" e "_self" Eles dirão se após clicar no botão enviar, se a página irá abrir em uma nova aba ou na mesma aba. _self = mesma aba. _blank = Uma nova aba. atributo autocomplete Possui os valores "on" e "off". Serve para o navegador gravar o usuário e senha no formulário já preenchido. Ou seja, quando você entrar novamente em um formulário já preenchido anteriormente, ele vai auto completar para você.
