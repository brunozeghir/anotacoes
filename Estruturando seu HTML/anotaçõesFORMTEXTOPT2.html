FORMATANDO TEXTOS - PARTE 2 tag fonte Modifica o tamanho da fonte da letra. Sozinha não faz nada, é necessário o uso de atributos. Atributo color: Usado para modificar a cor do texto. Atributo face: Colocamos no valor a fonte de letra que usaremos. Ex: Arial. É possível colocarmos mais de uma fonte, para caso um usuário não tenha a primeira fonte, o navegador vai tentar abrir a segunda fonte. Colocamos separados por vírgulas. Ex: face="Arial, Tahoma". A tag font está caindo em desuso pois é mais comum codificarmos a fonte através do CSS.
