DIV & SPAN Tag div Uma tag para delimitar espaços, ela não é exibida na tela para o usuário. uma tag tipo display block, q ocupa todo seu espaço horizontal. Tag span Ela n possuí display block. Mais serve para selecionar um trecho para podermos modificar via css e tal. Ex: Dio.me, se eu quiser mudar a cor só da palavra dio, eu coloco a tag span na palavra dio, e no css eu chamo a configuração em cima da tag span.
