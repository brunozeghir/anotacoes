FIELDSETS A tag fieldset cria como se fosse um retangulo, uma borda, em volta das tags que estão dentro dele. É uma tag mais visual. tag label, é como se fosse uma span. Mas ela sempre está junta de outra tag. Exemplo a label antes de uma input, é a label de uma tag input. Dentro da tag fieldset, vc pode colocar a tag legend com algo escrito dentro. Com isso, o que for escrito, vai aparecer entre as linhas da fieldset, como se fosse uma legenda mesmo para aquele quadrado de dados.
