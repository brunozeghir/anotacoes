TAG AUDIO Tag específica para colocarmos audios. A forma dele é assim. o atributo controls Ele vai colocar os botões de play, pause, volume... o atributo autoplay: Esse atributo serve para quando você abrir o site ele já começa a tocar o audio. Sem o usuário ter clicado. Caso eu tire o atributo controls e deixe o autoplay, vira um site com música de fundo. Pois os controles para o usuário somem.
